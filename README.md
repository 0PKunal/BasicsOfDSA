A clean, beginner-focused C repository for learning core Data Structures and Algorithms through small, single-purpose programs.

## Overview

This project is organized by topic so you can practice one concept at a time.
Each folder includes practical files for common operations such as:

- Input/creation
- Insertion
- Deletion
- Reversing
- Searching

Sorting algorithms are grouped in a separate folder.

## Topic Map

- Arrays: Basic array operations and traversal logic.
- Linked Lists: Fundamental node-based operations.
- Queues: FIFO operations and queue handling patterns.
- Sorting: Elementary sorting algorithms.
- Stacks: LIFO operations and stack handling patterns.
- Trees: Introductory tree operations.

## Prerequisites

- GCC compiler installed and available in PATH
- VS Code (recommended)

## Build And Run

###  Command Line

Compile and run any file directly:

```powershell
gcc Arrays/InputArray.c -o InputArray.exe
.\InputArray.exe
```
## Learning Path (Suggested)

1. Arrays
2. Stacks
3. Queues
4. Linked Lists
5. Trees
6. Sorting

## Goal Of This Repository

Provide simple, readable C implementations to strengthen DSA fundamentals before moving to advanced problem solving.
